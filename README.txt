Pokemon Colosseum - author: Pamella Barbosa Viana

The contents of this project are:
moves-data.csv
pokemon-data.csv
Pokemon.py
Move.py
PokemonColosseum.py (where main() is)


The only library that needs to be installed in order to run this project is NumPy. 

Here is the link that goes over how to install it: https://numpy.org/install/


How to execute the project:

- Make sure python 3.11 is installed. (If not, you can find how to install it here: https://www.python.org/downloads/)
- Run the following command in your favorite terminal:
  
python3 PokemonColosseum.py
